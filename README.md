**🔹 UC10 – Count Contacts by City or State** Branch: feature/uc10-CountContactsByCity

🎯 Objective

- Maintain a count of contact persons categorized by City or State
- Use Java Streams for efficient data aggregation across the system

✅ Implementation

- Leveraged Java Streams `filter` and `count()` operations to calculate totals
- Implemented logic to aggregate contact counts across multiple Address Books in the system
- Provided functionality to display the total number of contacts for a specific City or State as per user request
- Optimized the counting process to ensure accurate and instant statistical results
