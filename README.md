**🔹 UC6 – Manage Multiple Address Books** Branch: feature/uc6-MultipleAddressBooks

🎯 Objective

- Support unique Address Books for different categories (e.g., Personal, Professional)
- Organize multiple Address Books within a single system using a dictionary

✅ Implementation

- Introduced a `HashMap<String, AddressBook>` to store multiple Address Books with unique names
- Implemented logic to allow users to create a new Address Book or switch between existing ones
- Enabled functionality to perform CRUD operations on the currently selected Address Book
- Enhanced the system architecture to manage a collection of Address Books efficiently
