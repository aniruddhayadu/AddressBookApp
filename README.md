🎯 Objective

- Enhance interoperability with web applications and external systems

✅ Implementation

- Integrated the **GSON** (Google JSON) library into the project
- Implemented logic to serialize the entire Contact list into a structured `.json` file
- Developed deserialization methods to reconstruct the Address Book from JSON data accurately
- Ensured that complex list structures and object mappings are preserved during data conversion
