**🔹 UC11 – Sort Contacts Alphabetically by Name** Branch: feature/uc11-SortContacts

🎯 Objective

- Ensure contacts in the Address Book are sorted alphabetically by person's name
- Maintain a sorted list for better readability and searching

✅ Implementation

- Used Java Streams `sorted()` method with a custom Comparator
- Implemented sorting based on the `firstName` of the contacts
- Ensured the sorted order is maintained when displaying all entries
- Verified that new additions do not disrupt the overall sorting logic of the Address Book
